## Install modules
**Linux**: `./npm_pkgs.sh`  
**Windows**: `./npm_pkgs.bat`  
**MacOS** (untested): `./npm_pkgs.sh`
## Usage
To start the bot: `npm run dev`  
**Note**: you need to provide a `.env` file with the following contents: `tok=YOUR_BOT_TOKEN`; where `YOUR_BOT_TOKEN` is your app token
